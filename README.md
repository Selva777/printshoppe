# Print Shoppe - a 3D print queue for hackathons and cool conferences

## Install

1. ```git clone```
2. ```npm install```
3. Create file config.js from example, enter in your api details
4. ```node index.js```
5. open browser

paths are: 
+ localhost:8000/
+ localhost:8000/queue  
(queue requires github login)
